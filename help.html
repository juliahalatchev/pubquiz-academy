Så fungerar appen

Träningslägen

Question Engine väljer frågor adaptivt. Bishops Intelligence bygger bredare pubquizrundor. Visual Quiz använder flaggor, symboler och visuella ledtrådar. Question Factory låter dig redigera frågebanken lokalt.

Statistik

Appen sparar dina svar, felsvar, kategorier, repetitionsintervall och senaste rundor lokalt i din webbläsare.

Backup

På startsidan kan du exportera en backupfil och senare importera den på samma eller en annan enhet.

Integritet

Ingen personlig statistik skickas till någon server. Allt lagras lokalt på din enhet. GitHub Pages levererar bara själva appfilerna.

Question Factory

Ändringar i adminverktyget sparas lokalt tills du exporterar en ny questions.json och publicerar den via GitHub Desktop.

Installera på hemskärmen

Öppna appen i Safari på iPhone, tryck på Dela och välj Lägg till på hemskärmen.
